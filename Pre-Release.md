# BubblesTheDev Web Browser 1.0.37 Pre-Release

This is a pre-release information page for BubblesTheDev Web Browser version `1.0.37`.

It gives a simple preview of what is coming in this build, especially the new Music Downloader feature.

## What This Pre-Release Is

This pre-release is an early look at the next browser update before the full release is finalized.

It is meant for people who want a clear, non-technical summary of what is being added and what the feature is designed to do.

## Main New Feature In 1.0.37

Version `1.0.37` introduces a new **Music Downloader** experience built around privacy, safety, and user control.

The goal is to make the feature feel simple and easy to use while still keeping strong limits in place behind the scenes.

## What The Music Downloader Is For

The Music Downloader is designed for downloading audio from supported YouTube video links.

It is meant to be:

- easy to understand
- local-first
- privacy-focused
- limited and controlled
- focused on single downloads instead of bulk use

## What Users Can Expect

The Music Downloader is designed around a more careful and restricted experience.

- It is focused on single supported video links only.
- It is made for audio downloads, not video downloads.
- It is intended to help reduce misuse by limiting how downloads can be started and processed.
- It includes queue and cooldown behavior to keep usage controlled.
- It is built to run locally on the device instead of acting like a cloud service.

## Safety And Control

This pre-release puts a strong focus on making the feature feel responsible and controlled.

- The downloader is not meant for bulk downloading.
- It is not meant for playlists, channels, mixes, albums, livestreams, or similar large-scale use.
- It is built with limits that help prevent repeated abuse or rapid request spam.
- Downloads are checked before they are finalized.
- The feature is designed to stay inside approved browser-controlled locations.

## Privacy

The browser continues to follow a privacy-first direction in this build.

- Everything is intended to run locally on the device.
- No built-in analytics are described for this feature.
- No tracking is described for this feature.
- The downloader is intended to avoid exposing private file details or sensitive user data.

## User Responsibility

Before using the Music Downloader, users are expected to acknowledge a simple responsibility notice.

This is meant to remind users to only download content they own or have permission to use.

## Browser Experience

The feature is planned to appear as part of the browser interface with a simple music area that includes:

- a URL entry box
- a download button
- status messages
- queue information
- progress feedback
- cooldown information when needed

## Important Note

Because this is a pre-release, details may still change before the final public release.

This page is only meant to share information in a simple way and does not include download or setup instructions.

## Quick Summary

BubblesTheDev Web Browser `1.0.37` pre-release focuses on a new Music Downloader feature that aims to be user-friendly, privacy-first, locally processed, and more carefully controlled than a typical unrestricted downloader.
