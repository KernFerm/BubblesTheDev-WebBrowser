# BubblesTheDev - WebBrowser: 1.0.25

- Security, Themes, Toolbar, and Update Improvements

## Highlights

- New built-in muted themes so users can switch the browser look without bright colors
- Toolbar show and hide controls, while keeping quick access available from the `»` menu
- Updated dropdown behavior for bookmarks and tools so menus feel cleaner and no longer leave empty space on the right
- New shortcut keys for bookmark bar, toolbar visibility, theme cycling, and the updated browser tools
- Stronger update safety with secure update-link checks and SHA-256 verification before installer updates run
- Safer extension importing with extra checks, warnings for higher-risk permissions, and no local file access by default
- Password save and reveal protections now stay limited to secure website contexts
- Cleaner browser shell handling with tighter page protections and safer renderer updates behind the scenes
- Existing browser features remain in place, including saved passwords, bookmark import, extension import, VPN tools, split view, site permissions, ad blocking, download protection, sleepable tabs, and runtime diagnostics

## What's New In 1.0.25

- Users can now choose from several built-in muted themes, including Graphite, Slate Blue, Forest Ash, and Mocha.
- The toolbar can now be shown or hidden without removing quick access to important browser tools.
- The `»` quick menu stays available even when the toolbar is hidden.
- Bookmarks and tools dropdown behavior was improved so those menus no longer create the empty box area on the right side of the browser shell.
- New keyboard shortcuts were added for bookmark-bar visibility, toolbar visibility, and theme cycling, and the shortcut help list was updated to match.
- Managed updates are now checked more carefully and only continue when secure links and a matching SHA-256 are present.
- Imported extensions now go through stricter checks before loading and show extra warnings when they request broader access.
- Password saving and password reveal actions now stay limited to secure website contexts for better protection.
- Browser shell protections were tightened with safer content rules and renderer cleanup work.
- General polish and stability updates were included across the browser shell and release documentation.
