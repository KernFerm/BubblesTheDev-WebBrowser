## Current Release

- Version `1.2.110` is the current release.
- Older versions are not supported for current fixes and improvements.

Current documentation highlights for this release:

- version `1.2.110` is the current public browser release reflected in the docs
- browser profiles, Guest Mode, optional profile PIN protection, and profile recovery controls remain part of the current browser build
- encrypted profile secrets, session snapshots, restore points, and profile backup or restore behavior remain part of the local-first browser model
- the multilingual foundation still includes `679` supported locale packs and `453` base language families
- the browser keeps its local-first privacy posture with no built-in telemetry or analytics
- the shell theme menu now includes additional built-in themes, the animated `Aurora Drift` theme, and a custom local-image banner theme option
- the older `Casual` theme is no longer part of the current built-in theme list
- Bubbles Search Engine now blends DuckDuckGo and Google results on `bubbles://home` and labels which provider supplied each result
- bundled `yt-dlp.exe` is now `2026.07.04`, and bundled `ffmpeg.exe` / `ffprobe.exe` are now `8.1.2`
- update downloads can still be handled automatically by the browser, and the end user completes installation through the regular installer flow
- after the browser downloads the new version, the regular installer opens and the end user goes through the installer prompts
- normal installed browser clients can still check for updates and check in without exposing an embedded shared update secret in the shipped app files
- the browser keeps its current privacy, accessibility, localization, and profile feature set in place

For the latest browser announcements, release posts, and community updates, check the Discord server above.
