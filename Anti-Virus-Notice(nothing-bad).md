# ⚠️ Antivirus / SmartScreen Notice

Some antivirus programs (such as Norton, Windows Defender SmartScreen, etc.) may flag or block the installer for **BubblesTheDev Web Browser**.

## 🧠 Why This Happens

This is **NOT** because the application is malicious.

It happens because:

- The application is **new**
- It does not yet have a long **download reputation**
- It is **not code-signed** (yet)
- Antivirus systems are **overly cautious** with unknown software

This is very common for independent developers and new releases.

---

## 🛠️ What To Do If It Gets Blocked

### 🔹 Windows SmartScreen
If you see:
> “Windows protected your PC”

1. Click **More info**
2. Click **Run anyway**

---

### 🔹 Norton (or similar antivirus)

If the file is blocked or removed:

1. Open your antivirus (e.g., Norton)
2. Go to **Security History**
3. Find the blocked installer
4. Click **Options**
5. Select **Restore and allow**

---
